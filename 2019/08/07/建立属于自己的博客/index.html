建立属于自己的博客

2019-08-07 • 分类: GitHub • 标签: hexo, 学习

前言：
这是我第一次在GitHub上完成的个人博客。希望它能一直激励我前进，故取名为学习之路，我用它来记录我学习的历程以及遇到的问题。于是第一篇便用来记录我建立博客的过程。

准备工作

在做博客之前应该先了解有哪些建立的方法，找一个适合自己的方法，我使用的方法就是用GitHub里的博客框架hexo来做，hexo使用markdown来分析文章，简单快捷，最主要还是用GitHub就可以免费建立静态网站不用另外租服务器。

1、安装git、node.js和hexo

git

分布式版本管理系统，也就是管理文件不同版本的系统。
官方网站：https://gitforwindows.org/
可能因为服务器在国外的原因，下载慢的出奇，于是我选择在github下载，下载安装完后直接安装。

node.js

在服务器上跨平台的JavaScript执行环境。
官方网站：https://nodejs.org/en/

node.js -v #查看版本

在安装完node.js后在cmd里输入指令把npm（node包的管理器）替换成淘宝镜像，安装插件会变快。

2、搭建GitHub

创建仓库

在GitHub中点击New repository建立，名字格式为：yourname.github.io。
InInitialize this repository with a README选项可勾。
回到cmd控制台 （设置github用户名 和邮箱）

创建、设置SSH

在生成SSH的文件夹（默认为C:\Users\Administrator.ssh）找到id_rsa.pub文件复制里面的所有内容。
在GitHub网站里进入setting-SSH and GPG keys添加新的ssh key。title随便取，把刚才复制的部分放进key里。
